# Placement System API

Spring Boot backend project for managing student placements.

## Features
- Add student
- Get student by ID
- Get all students
- Delete student
- Validation
- Global exception handling

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## How to Run
1. Clone the repo
2. Update MySQL config in application.properties
3. Run PlacementSystemApplication.java
4. Use Postman to test APIs
